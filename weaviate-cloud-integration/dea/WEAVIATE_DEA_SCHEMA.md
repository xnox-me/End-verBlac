# Weaviate Schema for DEA Project

This document defines the Weaviate schema for the DEA (Decentralized Exchange Analytics) project.

## Classes

### 1. TradingPair
```javascript
const tradingPairClass = {
  class: 'DEATradingPair',
  description: 'A trading pair in the DEA platform',
  vectorizer: 'none',
  properties: [
    {
      name: 'symbol',
      dataType: ['string'],
      description: 'The trading pair symbol (e.g., ETH/USDT)',
    },
    {
      name: 'baseAsset',
      dataType: ['string'],
      description: 'The base asset of the trading pair',
    },
    {
      name: 'quoteAsset',
      dataType: ['string'],
      description: 'The quote asset of the trading pair',
    },
    {
      name: 'chain',
      dataType: ['string'],
      description: 'The blockchain network',
    },
    {
      name: 'price',
      dataType: ['number'],
      description: 'Current price of the trading pair',
    },
    {
      name: 'volume24h',
      dataType: ['number'],
      description: '24-hour trading volume',
    },
    {
      name: 'timestamp',
      dataType: ['date'],
      description: 'Timestamp of the data',
    },
  ],
};
```

## Integration Example

```javascript
// Example of initializing DEA schema
const weaviateService = require('./weaviateService');

const initializeDEASchema = async () => {
  const classes = [tradingPairClass, tradeClass, analyticsReportClass];
  await weaviateService.initializeSchema(classes);
};

// Example of adding a trading pair
const addTradingPair = async (data) => {
  await weaviateService.addData('DEATradingPair', data);
};

// Example of semantic search for trading pairs
const searchTradingPairs = async (query) => {
  return await weaviateService.semanticSearch('DEATradingPair', query);
};
```
